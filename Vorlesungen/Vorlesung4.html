Wie der Code zeigt gibt es verschiedene Möglichkeiten, wie eine Art Gruppe gebildet werden kann, auf die der Style zutreffen soll. Das erste Beispiel ist dass man mit CSS direkte Tags ansprechen kann. Somit wird in unserem beispiel festgelegt, dass für html, also für die gesamte Seite, das smoothe scrolling aktiviert werden soll. Ab Zeile 8 hingegegen wird durch .Fusszeile eine Klasse erstellt, die unter anderem die Textfarbe schwarz und die Hintergrund Farbe weiss haben soll. Somit kann dieser definierte Style gezielt eingesetzt werden. Dcoh wie funktioniert das? Anhand des .Fusszeile-Beispiel funktioniert es indem man einem div-Tag class="Fusszeile" mitgibt. Somit folgt alles was innerhalb des div-Tags steht den definierten Anforderungen für .Fusszeile, außer innerhalb des Tags wird nochmal ausdrücklich für bestimmte Stellen etwas anderes definiert. Falls dies der Fall ist folgen diese Stellen dann primär den individuellen Angaben und sekundär den Angaben die in .fusszeile definiert wurden.
